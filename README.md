A lightweight library for encoding and decoding JSON with comments and trailing commas support.

```dart
import 'package:jsonc/jsonc.dart';
```

```dart
final json = jsonc.decode('''
{
  // one-line comment
  "foo": "bar",
  /*
   * multi-line block comment
   */
  "baz": "qux",
}
''');
print(json); // {foo: bar, baz: qux}
```
